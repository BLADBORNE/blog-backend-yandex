# Blog Backend

Бэкенд-приложение для блога.

---

## Описание

Проект использует PostgreSQL с Testcontainers в интеграционных тестах.  
Для запуска приложения необходимо включить Docker.

---

## Сборка и запуск

1. Собрать проект:

```bash
mvn clean package
```
2. Запустить приложение через Docker Compose docker-compose.yaml
