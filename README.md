# Blog Backend

Бэкенд-приложение для блога.

---

## Описание

Проект использует PostgreSQL с Testcontainers в интеграционных тестах.  
Для запуска приложения необходимо включить Docker.

---

## Сборка и запуск

1. Запустить базу данных через Docker Compose docker-compose.dev.yaml
2. Запустить проект через в ide во вкладке configurations
